## 1. 合规风险分级

| 等级 | 场景 | 需要资质 | 风险 |
|---|---|---|---|
| 🟢 低 | **自用**看行情 + 跑回测 | 无 | 无 |
| 🟡 中 | **小范围分享**（亲友同事） | 视情况 | 灰色 |
| 🔴 高 | **ToC 公开下载** | 需"金融信息服务业务"资质 | 监管下架 |
| 🔴 极高 | **代客理财 / 承诺收益** | 需"证券投资咨询业务" + "私募基金"牌照 | 刑事 |

## 2. 项目定位

灵眸量化 v0.1 = **个人量化研究工具**，定位 🟢 低风险。

### 2.1 允许做的事 ✅
- 自用看 A 股 / 港股 / 美股行情
- 跑历史数据回测，验证自己的想法
- 写自己的策略代码
- 学习量化交易知识
- **模拟盘**实盘测试（不真金白银）
- 数据导出做研究

### 2.2 不允许做的事 ❌
- ❌ "承诺收益" / "保证盈利" 字样
- ❌ "代客理财" / "代为操作账户"
- ❌ "非法荐股" / 推荐个股 / 收费荐股
- ❌ 公开宣传"无风险高收益"
- ❌ 用本项目做"投资顾问"业务

### 2.3 灰色地带 ⚠️
- 朋友圈分享自己的回测截图 → 没问题
- 把工具给小范围亲友用 → **不宣传、不收费**，问题不大
- 发布到 GitHub 开源 → 注意 README 加合规声明
- 做"策略市场"（买卖策略）→ 涉及"金融信息增值服务"，需资质

## 4. 实盘交易合规（V1.1 启用）

### 4.1 必须持牌
- **证券投资咨询业务**: 投资建议、荐股
- **私募基金管理人**: 代客理财
- 灵眸量化本身**不做这两类业务**

### 4.2 用户自行实盘
- 用户用自己的券商账户接入 → 用户自负盈亏
- 灵眸量化仅提供**工具**，不构成投资建议
- 用户须知：在 `docs/用户协议.md` 明确说明

### 4.3 风险揭示（必须在 UI 显著位置）
- "本工具仅供学习研究，不构成投资建议"
- "历史回测不代表未来收益"
- "实盘交易有亏损本金风险，请谨慎决策"

## 6. 开源合规

### 6.1 LICENSE 选择
- 推荐 **MIT**（最宽松）
- 或 **AGPL-3.0**（要求衍生作品也开源，防止白嫖商用）

### 6.2 第三方依赖合规
- 主力依赖均为 MIT/BSD/Apache 2.0
- TA-Lib: BSD

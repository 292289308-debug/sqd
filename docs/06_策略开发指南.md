## 1. 策略 = 一个 Python 类

最简单的策略长这样：

```python
from app.services.strategy import Strategy, Context

class DualMA(Strategy):
    """双均线策略: MA5 上穿 MA20 买入, 下穿卖出"""
    
    params = {
        "short": 5,    # 短期均线周期
        "long": 20,    # 长期均线周期
    }
    
    def on_bar(self, ctx: Context):
        # ctx.today: 今日 K 线 (dict)
        # ctx.history: 截至今日的历史 K 线 (DataFrame)
        # ctx.position: 当前持仓数量 (int)
        # ctx.cash: 当前可用资金 (float)
        
        close = ctx.history['close']
        ma5 = close.rolling(self.params['short']).mean().iloc[-1]
        ma20 = close.rolling(self.params['long']).mean().iloc[-1]
        
        if ma5 > ma20 and ctx.position == 0:
            # 金叉: 全仓买入
            qty = int(ctx.cash / ctx.today['close'] / 100) * 100  # 整手
            ctx.buy(qty)
        elif ma5 < ma20 and ctx.position > 0:
            # 死叉: 清仓
            ctx.sell(ctx.position)
```

把这个类保存到 `data/strategies/user_xxx.py`，提交回测即可。

## 2. Context API

| 属性 / 方法 | 类型 | 说明 |
|---|---|---|
| `ctx.today` | dict | 当日 K 线 {open, high, low, close, vol} |
| `ctx.history` | DataFrame | 截至今日的全部历史 K 线 |
| `ctx.position` | int | 当前持仓股数 |
| `ctx.cash` | float | 可用资金 |
| `ctx.portfolio_value` | float | 当前总资产（现金 + 持仓市值）|
| `ctx.buy(qty, price=None)` | method | 买入（默认市价） |
| `ctx.sell(qty, price=None)` | method | 卖出 |
| `ctx.indicator(name, fn, *args)` | method | 自定义指标 |

类通达信公式：

```
MA1: MA(CLOSE, 5);
MA2: MA(CLOSE, 20);
金叉: CROSS(MA1, MA2);

```

## 6. 回测报告解读

```json
{
  "total_return": 0.0084,        // 总收益 (8.4%)
  "annual_return": 0.0369,       // 年化收益 (3.69%)
  "benchmark_return": 0.0003,    // 基准收益（同期持有）
  "alpha": 0.0366,               // 超额收益
  "sharpe_ratio": 0.69,          // 夏普比率 (年化)
  "max_drawdown": -0.0184,       // 最大回撤 (-1.84%)
  "trade_days": 58,              // 交易日数
  "trade_count": 3,              // 交易笔数
  "win_rate": 0                  // 胜率 (V1.0 实现 buy-sell 配对)
}
```

### 6.1 关键指标判断

| 指标 | 优秀 | 一般 | 警惕 |
|---|---|---|---|
| 年化收益 | > 20% | 10~20% | > 50%（可能过拟合）|
| 夏普比率 | > 1.5 | 1.0~1.5 | < 0.5 |
| 最大回撤 | < 10% | 10~20% | > 30% |
| 胜率 | > 55% | 40~55% | < 30% |
| Alpha | > 5% | 0~5% | < 0 |

### 6.2 过拟合识别
- 回测收益 > 实盘预期收益 2 倍以上 → 警惕
- 换一只股票 / 换一段时间就失效 → 过拟合
- 调参后收益大增 → 过拟合（参数孤岛）
- **应对**: Walk-Forward / 蒙特卡洛 / 样本外验证

## 7. 实战示例

### 7.1 海龟交易
```python
class Turtle(Strategy):
    params = {"entry": 20, "exit": 10}
    
    def on_bar(self, ctx):
        high = ctx.history['high']
        low = ctx.history['low']
        close = ctx.today['close']
        
        # 20 日新高买入
        if close > high.iloc[-self.params['entry']:].max() and ctx.position == 0:
            qty = int(ctx.cash / close / 100) * 100
            ctx.buy(qty)
        # 10 日新低卖出
        elif close < low.iloc[-self.params['exit']:].min() and ctx.position > 0:
            ctx.sell(ctx.position)
```

### 7.2 布林带回归
```python
class BollingerRevert(Strategy):
    params = {"period": 20, "std": 2}
    
    def on_bar(self, ctx):
        close = ctx.history['close']
        mid = close.rolling(self.params['period']).mean()
        std = close.rolling(self.params['period']).std()
        lower = mid - self.params['std'] * std
        upper = mid + self.params['std'] * std
        
        price = ctx.today['close']
        if price < lower.iloc[-1] and ctx.position == 0:
            ctx.buy(int(ctx.cash / price / 100) * 100)
        elif price > upper.iloc[-1] and ctx.position > 0:
            ctx.sell(ctx.position)
```

### 7.3 多因子选股（V1.0）
```python
class MultiFactor(Strategy):
    """每月按动量 + 价值因子打分, 选 Top 5"""
    params = {"top_n": 5, "lookback": 60}
    
    def on_init(self):
        self.universe = self.params.get('universe', [])  # 股票池
    
    def on_bar(self, ctx):
        if ctx.today_index % 20 != 0:  # 每月调仓
            return
        
        scores = {}
        for code in self.universe:
            hist = ctx.get_history(code)
            momentum = hist['close'].iloc[-1] / hist['close'].iloc[-self.params['lookback']] - 1
            scores[code] = momentum
        
        top = sorted(scores, key=scores.get, reverse=True)[:self.params['top_n']]
        ctx.rebalance({code: 1.0 / len(top) for code in top})
```

## 8. 防过拟合工具（V1.0）

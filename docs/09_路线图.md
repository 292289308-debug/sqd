## Phase 1: MVP 看盘+回测 (2 个月)

**目标**: 跑通行情展示 + 量化回测的完整闭环

### M1 (Week 1-2): 数据接入
- [ ] 接 Tushare Pro 真实数据（5 年 A 股日线 + 分钟线）
- [ ] 港美股数据 (AkShare / Yahoo Finance)
- [ ] 增量同步 + Celery 定时任务
- [ ] 数据质量校验（缺失/异常/复权）

### M2 (Week 3-4): 看盘功能
- [ ] 自选股多分组 + 拖拽排序
- [ ] 多周期切换 (1m/5m/15m/30m/60m/1d/1w)
- [ ] 技术指标叠加 (MA/MACD/RSI/BOLL/KDJ)
- [ ] 分时图 + 盘口十档（Level-2 需采购）
- [ ] 异动提醒（涨跌幅 > 5% / 1 分钟）
- [ ] WebSocket 实时推送

### M3 (Week 5-6): 回测引擎
- [ ] 事件驱动回测（替代 backtrader 自研）
- [ ] T+1 撮合 + 滑点 + 手续费精确模拟
- [ ] 30+ 策略模板（双均线/海龟/布林带/多因子）
- [ ] 自定义公式系统（类通达信）
- [ ] 回测报告（图表 + 绩效归因）
- [ ] 防过拟合（Walk-Forward / 蒙特卡洛 / 样本外）

### M4 (Week 7-8): 用户系统 + 收尾
- [ ] 注册/登录（JWT）
- [ ] 多用户隔离（user_id 维度）
- [ ] 数据导入/导出（CSV）
- [ ] Docker 一键启动验证
- [ ] 文档完善（部署/运维/API）
- [ ] 内测（100 种子用户）

**里程碑**: 100 种子用户内测

## 商业化路径

### 短期（Phase 1 + V1.1）
- 用户增长为主，**不收费**
- 通过 GitHub / 雪球 / 知乎 推广
- 收集种子用户反馈

### 中期（V1.2 + Phase 2）
- **个人专业版** ¥99/月：完整功能 + 实时行情
- **团队版** ¥999/月：多账户 + API 访问
- 机构版按需询价

### 长期（盈利后）
- 策略市场抽佣（30%）
- 私募 SaaS（年付 ¥5~50 万）
- 教育版（学生免费，认证收费）

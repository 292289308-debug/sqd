10. `orders` — 订单

## 5. 数据流

### 5.1 看行情
```
前端 axios.get('/api/v1/kline/?ts_code=...')
    ↓
API 网关验证 + 限流
    ↓
KlineService 查询 TimescaleDB hypertable
    ↓
Redis 缓存（5s）
    ↓
返回 JSON
```

### 5.2 跑回测
```
前端 POST '/api/v1/backtest/run' {strategy_id, dates, universe}
    ↓
Backtest Engine (事件驱动)
    ├─ 加载 K 线 (TimescaleDB 或 CSV 缓存)
    ├─ 计算指标 (TA-Lib)
    ├─ 生成信号 (策略代码)
    ├─ 撮合 (T+1, 滑点, 手续费)
    ├─ 计算净值曲线
    └─ 统计指标 (夏普/回撤/Alpha)
    ↓
写回 backtest_runs 表
    ↓
返回完整结果
```

### 5.3 实时行情（V1.0）
```
Tushare WebSocket → QuoteService (asyncio)
    ↓
Redis Pub/Sub (channel: quote:{ts_code})
    ↓
FastAPI WebSocket → 前端 (Lightweight Charts subscribe)
```

## 6. 部署架构

### 6.1 本地开发（当前）
```
backend/  → python -m uvicorn app.main:app --port 8000
frontend/ → python -m http.server 5173 (CDN 版)
           或 npm run dev (Vite 版)
data/     → 本地 CSV 缓存 + SQLite
```

### 6.2 Docker Compose（生产化目标）
```yaml
services:
  timescaledb:  # 时序数据库
  redis:        # 缓存
  backend:      # FastAPI
  frontend:     # Vue 3 (Nginx)
  prometheus:   # 监控 (profile: monitoring)
```

### 6.3 Kubernetes（V2.0）
- 5 个微服务拆分：quote / strategy / backtest / news / broker
- Helm Charts
- HPA 自动扩缩容
- ArgoCD GitOps

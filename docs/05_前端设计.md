# 前端设计

**版本**: v0.1  
**框架**: Vue 3 + Vite + TypeScript  
**UI 库**: 自研深色金融风（Element Plus 备选）

---

## 1. 设计原则

1. **专业深色配色** — 金融从业者习惯深色（彭博 / 同花顺 / 通达信）
2. **信息密度高** — 一屏看更多数据（K 线 + 指标 + 自选股）
3. **响应快** — K 线渲染 < 500ms，操作无感延迟
4. **键盘友好** — 快捷键支持（自选股切换 / 周期切换）

## 4. 核心组件

### 4.1 KLineChart.vue
包装 TradingView Lightweight Charts

```vue
<template>
  <div ref="chartContainer" :style="{height: height}" />
</template>

<script setup lang="ts">
import { createChart, CandlestickSeries } from 'lightweight-charts'
const props = defineProps<{ data: KLine[] }>()
const chartContainer = ref<HTMLDivElement>()
let chart: IChartApi

onMounted(() => {
  chart = createChart(chartContainer.value!, { ... })
  const series = chart.addSeries(CandlestickSeries, { ... })
  series.setData(props.data.map(toTVDatetime))
})
</script>
```

### 4.2 StockTable.vue
通用股票列表（带排序 / 筛选 / 搜索）

### 4.3 EquityChart.vue
包装 ECharts 展示净值曲线

### 4.4 IndicatorPicker.vue
技术指标选择器（MA / MACD / RSI / BOLL / KDJ）

## 6. MVP 版（CDN 单文件）

为了 0 构建直接演示，Phase 1 用 CDN 单文件：

```html
<script src="https://unpkg.com/vue@3.4.27/dist/vue.global.prod.js"></script>
<script src="https://unpkg.com/axios@1.7.2/dist/axios.min.js"></script>
<script src="https://unpkg.com/lightweight-charts@4.1.3/dist/lightweight-charts.standalone.production.js"></script>
<script src="https://unpkg.com/echarts@5.5.0/dist/echarts.min.js"></script>
```

优势：
- ✅ 0 构建，浏览器直接打开
- ✅ 部署简单（任何静态服务器）
- ✅ 改完代码立即生效
- ❌ 不易扩展（单文件 < 500 行后维护困难）

V1.0 迁移到 Vite + 组件化。

## 8. 关键交互

### 8.1 K 线图
- 鼠标滚轮：缩放
- 鼠标拖拽：平移
- 双击：复位
- 十字光标：显示 OHLC + 成交量
- 周期切换：1m / 5m / 15m / 30m / 60m / 1d / 1w

### 8.2 回测表单
- 实时校验（股票代码合法 / 日期范围 / 资金 > 0）
- 一键填充示例数据
- 提交后转圈 + 进度条
- 完成后滚动到结果区

### 8.3 自选股
- 拖拽排序
- 分组管理（新建 / 删除 / 重命名）
- 批量导入（粘贴代码列表）
- 一键全屏 K 线

## 10. 性能优化

- K 线数据懒加载（首次只加载 60 日）
- 滚动加载更多（向上滚动）
- 长列表虚拟滚动（自选股 > 100 时）
- WebSocket 推送时只更新最新一根 K 线（不重绘整图）
- 防抖（搜索输入 300ms）
- Vite 构建分包（vendor / charts / app）

---

## 11. 可访问性 (a11y)

- 键盘导航（Tab / Enter / 方向键）
- ARIA 标签
- 颜色对比度 ≥ 4.5:1
- 不依赖颜色传达信息（涨/跌同时用箭头和颜色）
